release notes

version history

v0.2.2

optional screen audio
  • added a share computer audio option for screen sharing.
  • enabled windows desktop system-audio capture when the option is selected.
  • updated remote playback so microphone audio and shared screen audio can play separately.

v0.2.1

windows screen sharing fix
  • fixed screen sharing inside the windows desktop app by adding electron display capture handling.
  • added a native desktop screen selection prompt before sharing starts.
  • kept browser screen sharing behavior unchanged.

v0.2.0

desktop distribution update
  • added desktop auto-update checks through github releases.
  • added the stable installer download named schibbs-mic-setup.exe.
  • added updater assets for installed desktop apps.
  • added clearer microphone, camera, screen sharing, server, and network errors.
  • added local font selection in the lobby and inside rooms.

v0.1.0

first windows installer
  • added the first github-built windows installer.
  • connected the desktop app to the render-hosted browser app.
  • published the first downloadable release asset.

current app

core room experience
  • voice rooms, text tab, camera, and screen sharing up to requested 1080p 60fps with optional computer audio when supported.
  • youtube music bot commands with play, pause, resume, and stop.
  • light and dark mode, color themes, fullscreen video viewing, and local appearance preferences.
