about

schibb's mic is a browser and windows voice-room app for small rooms, text chat, camera, screen sharing, and shared youtube music playback.

the windows app opens the same hosted room system as the browser app, so users can join each other from either place.

rooms

create or join a room with a name, then use voice, text, camera, and screen sharing in one focused space.

desktop

windows users can install the desktop app, receive update checks, and keep using the hosted render service.

preferences

theme, color, and font choices stay local on each device, so no member database is required for them.
